----------Условие----------
Напишите функцию на языке Golang, которая находит самую длинную строку общего префикса среди массива строк.

Если общего префикса нет, вернуть пустую строку "".

----------Решение----------
func longestCommonPrefix(strs []string) string {
    if len(strs) == 0 {
        return ""
    }
    prefix := strs[0]
    for i := 1; i < len(strs); i++ {
        for len(prefix) > 0 && !strings.HasPrefix(strs[i], prefix) {
            prefix = prefix[:len(prefix)-1]
        }
        if len(prefix) == 0 {
            return ""
        }
    }
    return prefix
}
Функция начинает с первой строки в массиве и сравнивает ее с каждой другой строкой, обрезая общий префикс, если он находится.
Если в какой-то момент не найдено совпадение или префикс сокращается до пустой строки, функция возвращает пустую строку.
Если все строки имеют общий префикс, функция возвращает этот префикс.
